# Licensing

No final open-source license has been selected in this cleaned AIR paper-code tree.

Before publishing or redistributing this repository, the project owner should choose and add an explicit license file, for example `LICENSE`, after confirming compatibility with:

- The AIR paper authors' intended release terms.
- Third-party code retained from the local `advDF` tree.
- Linked target-model repositories such as SimSwap, FaceShifter, MegaGAN, and AgileGAN.
- Vendored or required dependencies.
- Model checkpoints and datasets, which may have separate licenses or access terms.

Until an explicit license is added, treat this tree as an internal research preparation artifact rather than a fully licensed public release.

Recommended release action:

1. Choose a license with the project owner.
2. Add a root-level `LICENSE` file.
3. Update this file with the chosen license name.
4. Re-run `tools/release_check.py` and `tools/check_paper_alignment.py`.
